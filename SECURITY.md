## Reporting a Vulnerability

Use this section to tell people how to report a vulnerability.

If you find a vulnerability in the Phantom Logic Bridge or the Omni-Node manifest, do not open a public issue. Please follow these steps:

Submit a Report: Send a detailed log of the terminal-output or the failing script_3.js logic to the lead developer, PresidentAdam52.

Acknowledgement: You can expect a response regarding the Handshake Test within 24 hours.

Evaluation: If the vulnerability is accepted, the system will initiate a Global Sync to push a fix to all 1,200 Warrior Nodes.

Stealth Protocol: If a report is declined, the data will be purged from the terminal-emulator for system security and GDPR anonymity.
